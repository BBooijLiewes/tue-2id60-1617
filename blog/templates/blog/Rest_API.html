Rest API

Any user of this website can make use of the build in REST API. A request can be made for both the title and content of either all blog posts or one specific blog post

The format of these Rest request is Get upon which an JSON object will be send back containing the specific request

Format

To request all posts simply go to:

https://tue-2id60-1617.herokuapp.com/api/all/?format=json

To request a specific blog post first get that blog posts ID via the URL in that posts detail view, it will be a number located behind the /post/.

After obtaining this number simply go to the following URL:

https://tue-2id60-1617.herokuapp.com/api/Post_ID_HERE/?format=json
